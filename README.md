# Neatly Uninstaller

Neatly Uninstaller is a simple Windows application to uninstall programs and clean leftover files.  

> [!IMPORTANT] 
> This project is in early development and currently not really usable

## Planned Features

- Uninstall installed Windows applications
- Scan for leftover files and folders after uninstall
- Scan for leftover registry keys after uninstaller
- Clean temporary files related to uninstalled programs
- Minimal UI
- (Windows UWP app Support)
- (Support for uninstalling Steam games)

## Requirements

- Windows 10 or newer
- .NET 10 or higher
